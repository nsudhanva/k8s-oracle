## Key Features

<CardGrid stagger>
	<Card title="Always Free" icon="rocket">
		Runs entirely on Oracle Cloud's free tier (Ampere A1 ARM64 instances).
	</Card>
	<Card title="GitOps Powered" icon="github">
		Managed by **Argo CD**. All infrastructure and applications are defined as code.
	</Card>
	<Card title="Modern Ingress" icon="seti:graphql">
		Uses **Envoy Gateway** (Gateway API) for advanced traffic management.
	</Card>
    <Card title="Automated DNS & TLS" icon="setting">
        **External DNS** (Cloudflare) and **Cert Manager** (Let's Encrypt) handle domains and certificates automatically.
    </Card>
</CardGrid>
