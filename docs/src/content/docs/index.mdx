## Why OKE on Oracle Cloud?

- **Managed Control Plane:** OKE Basic Cluster provides a free, fully managed Kubernetes control plane.
- **High Performance:** Ampere A1 Compute instances provide up to 4 OCPUs and 24GB RAM for worker nodes.
- **GitOps Ready:** Complete Argo CD setup with App-of-Apps pattern.
- **Secure:** OCI Vault integration for secrets and Gateway API for secure ingress.

## Features

| Feature | Implementation |
|---------|----------------|
| **Kubernetes** | OKE Basic Cluster (Managed) |
| **Infrastructure** | Terraform (OCI Provider) |
| **GitOps** | Argo CD |
| **Ingress** | Envoy Gateway + OCI Load Balancer |
| **DNS** | External DNS + Cloudflare |
| **TLS** | Cert Manager + Let's Encrypt |
| **Secrets** | OCI Vault + External Secrets Operator |
| **LLM** | Gemma 3 1B via llama.cpp |
| **Docs** | Astro + Starlight (This site) |

## Comparison

| | **OKE Basic** | **K3s on VM** |
|---|---|---|
| **Control Plane** | Managed (Free) | Self-managed on VM |
| **Worker Resources** | Full 4 OCPU / 24GB | Reduced by Control Plane |
| **Maintenance** | Oracle Managed | Manual Upgrades |
| **Storage** | OCI Block Volume (CSI) | Local Path / HostPath |
