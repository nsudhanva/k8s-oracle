<Aside type="caution" title="Deployment Removed">
Open WebUI has been **fully removed** from this cluster. All `argocd/apps/open-webui/` manifests, the OCI OIDC application, the `oidc-credentials` secret, and the `chat.k8s.sudhanva.me` DNS/listener entries have been deleted.
</Aside>

## Why It Was Removed

Open WebUI was deployed as a chat interface for the self-hosted Gemma LLM. When Gemma was removed (see [Gemma LLM](/architecture/gemma-llm)), Open WebUI lost its backend and was no longer useful.

AI interaction is now handled by **OpenClaw**, which provides:

- **Telegram bot** (@CoochiepieBot) — conversational AI accessible from anywhere
- **Web UI** at `https://claw.k8s.sudhanva.me` — browser-based control panel

See the [OpenClaw docs](/architecture/openclaw) for the current setup.
